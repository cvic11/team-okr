# ⛳ 여기에 claude.ai 스킬을 붙여넣으세요 (자리표시자)

이 파일은 **사용자가 claude.ai에서 만든 스킬**을 담을 빈 슬롯입니다.
아직 실제 내용이 없습니다 — 아래 둘 중 하나로 채우면 됩니다.

## 채우는 법

**방법 A — 그냥 내용 주기**
claude.ai에서 그 스킬의 SKILL.md 내용을 복사해 Claude에게 주면,
Claude가 이 파일을 실제 내용으로 교체하고 폴더 이름도 알맞게 바꿔 커밋합니다.

**방법 B — claude.ai에서 직접 내보내기**
1. claude.ai → Settings → Capabilities/Skills → 해당 스킬 **다운로드(zip/폴더)**
2. 압축을 풀어 이 폴더(`_your-claude-ai-skill/`)를 통째로 교체
3. 폴더명을 스킬 이름에 맞게 변경 (예: `my-writing-style/`)
4. 커밋 → 다음 설치부터 자동 포함

## 주의
- 폴더명(`name:`)은 소문자-하이픈 권장. 다른 스킬과 겹치지 않게.
- 슬롯을 채우기 전까지 이 스킬은 동작하지 않습니다(설명만 있는 빈 스킬).
